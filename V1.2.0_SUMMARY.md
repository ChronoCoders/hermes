## Features Implemented

### 1. File Splitting
- 50MB chunk size
- SHA-256 hash per chunk
- JSON manifest generation

### 2. Memory-Efficient Processing
- Streams data in 50MB pieces
- Handles files larger than RAM
- Automatic cleanup

### 3. Encryption Support
- Password-based encryption
- Multi-recipient encryption
- TTL expiration support

### 4. Integrity Verification
- Per-chunk SHA-256 hashing
- Full file hash verification
- Tamper detection

### 5. Progress Tracking
- Per-chunk progress display
- Step-by-step status
- Clear error messages

## Technical Details

### Chunk Manifest Structure
```json
{
  "original_filename": "video.mp4",
  "total_size": 5368709120,
  "chunk_size": 52428800,
  "total_chunks": 103,
  "file_hash": "abc123...",
  "chunks": [
    {
      "index": 0,
      "size": 52428800,
      "hash": "def456...",
      "encrypted_path": "/vault/video.chunk.001.enc"
    }
  ]
}
```

### Process Flow

**Send:**
1. Split file â†’ chunks
2. Encrypt each chunk
3. Upload chunks
4. Create + encrypt manifest
5. Upload manifest
6. Cleanup temp files

**Receive:**
1. Download manifest
2. Decrypt manifest
3. Download all chunks
4. Decrypt each chunk
5. Reassemble file
6. Verify integrity
7. Cleanup temp files
